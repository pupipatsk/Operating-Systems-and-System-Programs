# Operating Systems and System Programs
 2110313 Operating Systems and System Programs 2025 Course, Chulalongkorn University

## Multipass Commands

Access an Instance
```bash
multipass shell Ubuntu-OS
```

Exit an Instance
```bash
ubuntu@Ubuntu-OS:~$ exit
```

Stop an Instance
```bash
multipass stop Ubuntu-OS
```

List Instances
```bash
multipass list
```
